Configuracion del Dashboard SEO

Completa estos campos una sola vez. Se guardan en tu navegador y el dashboard funciona automaticamente desde ahora.

console.cloud.google.com → Credenciales → OAuth 2.0 → copia el Client ID
console.groq.com/keys → Create API Key
aistudio.google.com → Get API Key

Se guardan solo en este navegador. Nunca se envian a ningun servidor.

✅ Registrar cambio aplicado

Se guardara la fecha de hoy y las metricas actuales como punto de comparacion (T0). El sistema verificara el impacto a los 7, 15 y 30 dias.
